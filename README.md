# Gote Drink Sales

เว็บนี้สามารถเอาไปเปิดใช้งานออนไลน์ได้ และยังเก็บเมนู รายการท็อปปิ้ง และข้อมูลรับเงินไว้ต่อเนื่อง ไม่หายทุกครั้งที่รีสตาร์ตเครื่อง

## วิธีเอาขึ้นออนไลน์

แนะนำใช้ Render เพราะโปรเจ็กต์นี้มีการบันทึกข้อมูลลงไฟล์จริง ถ้าใช้บริการที่ไม่เก็บไฟล์ถาวร ข้อมูลจะหายได้

1. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub
2. ไปที่ Render แล้วเลือกสร้างเว็บจาก GitHub
3. ให้ Render อ่านไฟล์ `render.yaml`
4. กด deploy

หลัง deploy เสร็จ Render จะได้ลิงก์เว็บถาวรไว้เปิดจากมือถือหรือส่งให้ลูกค้าใช้ได้เลย

## ข้อมูลจะถูกเก็บไว้ที่ไหน

- ตอนเปิดในเครื่องตัวเอง: เก็บในไฟล์ `app-state.json`
- ตอนเปิดบน Render: เก็บในพื้นที่ถาวรที่ mount ไว้ใน `/opt/render/project/src/data/app-state.json`

ถ้ายังไม่มีไฟล์ข้อมูลตอนเปิดครั้งแรก ระบบจะใช้ข้อมูลตั้งต้นจาก `recovered-drink-state.json`

## ไฟล์สำคัญ

- `server.js` เซิร์ฟเว็บและรับส่งข้อมูล
- `app-state.json` ข้อมูลล่าสุดของร้าน
- `recovered-drink-state.json` ข้อมูลสำรองตั้งต้น
- `render.yaml` ค่าที่ใช้เอาขึ้น Render

## เปิดใช้ในเครื่องอีกครั้ง

1. เปิดโปรเจ็กต์นี้
2. ติดตั้งแพ็กเกจถ้ายังไม่เคยติดตั้ง
3. เปิดเว็บด้วยคำสั่ง `npm start`
4. เข้า `http://localhost:4173`

ถ้าต้องการเปิดจากมือถือที่อยู่ Wi‑Fi เดียวกัน ใช้ลิงก์ IP ของคอมแทน `localhost`
