# Gote POS

โปรเจ็กต์นี้ถูกย้ายมาเป็น Next.js + TypeScript + Tailwind CSS แล้ว โดยออกแบบให้ใช้งานบนมือถือเป็นหลัก เหมาะกับร้านเครื่องดื่มที่ต้องการแตะรับออเดอร์เร็ว ดูยอดง่าย และปรับเมนูได้ทันที

## ฟีเจอร์หลัก

- รับออเดอร์เครื่องดื่มและท็อปปิ้งในหน้าเดียว
- ปิดบิลพร้อมสรุปยอดรวมทันที
- ดูสรุปยอดรายวันแยกเมนูและท็อปปิ้ง
- เพิ่มเมนูใหม่ ปรับราคา เปลี่ยนสถานะเปิดขาย
- ตั้งค่าชื่อร้านและข้อมูลพร้อมเพย์

## ข้อมูลถูกเก็บที่ไหน

ข้อมูลจะถูกเก็บใน `localStorage` ของเบราว์เซอร์เครื่องที่ใช้งานอยู่

- เหมาะกับการใช้งานร้านเดียวบนอุปกรณ์หลักเครื่องเดิม
- ถ้าเปิดจากเครื่องใหม่ ข้อมูลจะไม่ตามไปด้วย
- ถ้าต้องการเก็บข้อมูลข้ามหลายเครื่อง ควรต่อฐานข้อมูลภายหลัง

## เริ่มใช้งานในเครื่อง

1. ติดตั้งแพ็กเกจ
2. เปิดโหมดพัฒนา
3. เข้าเว็บที่ `http://localhost:3000`

```bash
npm install
npm run dev
```

## Deploy

เหมาะกับ Vercel มากกว่าโครงเดิม เพราะตอนนี้เป็น Next.js เต็มรูปแบบแล้ว

1. push ขึ้น GitHub
2. import repo ใน Vercel
3. deploy ได้ทันที
